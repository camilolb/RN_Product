## Project Structure

Our project is created using yarn, jotai, useQuery, Eslint, prettier, clean code, firebase

The project structure is

```
project
│   README.md
└───src
|   |---core
        |---adapters
        |---di
        |---domain
        |---framework
        |---hooks
        |---services
|   |---mobile
        |---context
        |---mobile-ui
        |---navigation
        |---pages
│   └───shared

```

## Dependencies

- Node.JS
- Java
- Yarn
- Cocoapods
- XCode
- Android Studio

## First Steps

You should follow the steep to configure environment of (https://reactnative.dev/docs/environment-setup)

After that use ``yarn install``


## Generic Comands

- Mobile
  - `yarn run ios`
  - `yarn run android`
